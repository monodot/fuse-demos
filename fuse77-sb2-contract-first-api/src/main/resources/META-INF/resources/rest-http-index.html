REST API Level 0 - Red Hat Fuse

The REST API Level 0 booster demonstrates how to map business operations to a remote procedure call endpoint over HTTP using a REST framework. This mission corresponds to Level 0 in the Richardson Maturity Model.

This booster introduces the mechanics of interacting with a remote (exposed by Apache Camel) service using the HTTP protocol.

By using this Fuse booster, you can quickly prototype and flexibly design your REST API.

To demonstrate the behavior of this booster:

  • Click camel/greetings/{name} to execute an HTTP GET request on the camel/greetings/{name} endpoint.

    A new browser window opens and shows a response in JSON format with a payload of Hello, $name! where $name is the value of the {name} URL parameter from the HTTP GET request. By default, this is Jacopo.

  • Change the name in the URL from Jacopo to your name. Observe that the response reflects the name change.

Click API Swagger page to open the REST API's Swagger page.
